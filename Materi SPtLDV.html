BAB 2: SISTEM PERTIDAKSAMAAN LINEAR DUA VARIABEL

2.1 Pengertian Pertidaksamaan Linear Dua Variabel

Pertidaksamaan linear dua variabel adalah kalimat terbuka yang menggunakan tanda pertidaksamaan, seperti >, <, ≤ , atau ≥, dan mengandung dua variabel yang berpangkat satu.

Pertidaksamaan dengan bentuk:

ax + by ≥ c; ax + by > c; ax + by ≤; ax + by < c

2.2 Pengertian Sistem Pertidaksamaan Linear Dua Variabel

Sistem pertidaksamaan linear dua variabel adalah dua atau lebih pertidaksamaan linear dua variabel, yang dimana antara pertidaksamaan yang satu dengan yang lain saling berkaitan.

Gambar 21. Grafik Sistem Pertidaksamaan Linear Dua Variabel


Bentuk umum dari sistem pertidaksamaan linear dua variabel:
ax + by > c; ax + by ≥ c; ax + by < c; ax +by ≤ c dengan a, b, c ∈ R





2.3 Sifat-sifat Pertidaksamaan

Sebelum memahami sistem pertidaksamaan linear, kita harus terlebih dahulu memahami sifat-sifat dari pertidaksamaan itu sendiri.

  1. Sifat tak negatif Untuk a ∈ R maka a ≥ 0.


  2. Sifat transitif
    3. Untuk a, b, c ∈ R jika a < b dan b < c maka a < c;

    jika a > b dan b > c maka a > c

  3. Sifat penjumlahan
    4. Untuk a, b, c ∈ R jika a < b maka a + c < b + c
    jika a > b maka a + c > b + c
    Jika kedua ruas pertidaksamaan dijumlahkan dengan bilangan yang sama, tidak mengubah tanda ketidaksamaan.

  4. Sifat perkalian
    5. Jika a < b, c > 0 maka ac < bc
    Jika a > b, c > 0 maka ac > bc
    Jika a < b, c < 0 maka ac < bc

    Jika kedua ruas pertidaksamaan dikalikan bilangan (riil) positif tidak akan mengubah tanda ketidaksamaan, sedangkan jika dikalikan bilangan negatif akan mengubah tanda ketidaksamaan.

  5. Sifat kebalikan
    Jika a > 0 maka 1/a> 0
    Jika a < 0 maka 1/a < 0




2.4 Himpunan Penyelesaian Pertidaksamaan Linear Dua Variabel

Himpunan penyelesaian dari pertidaksamaan dapat digambarkan pada garis bilangan, khususnya untuk himpunan penyelesaian berupa interval. Batas-batas interval digambarkan dengan menggunakan tanda (bulatan penuh) atau (bulatan kosong). Tanda bulatan penuh menunjukkan bilangan tersebut termasuk ke dalam himpunan penyelesaian, dan sebaliknya tanda bulatan kosong menunjukkan bilangan tersebut tidak termasuk ke dalam himpunan penyelesaian. (Gumilar, 2008, p. 69)

Gambar 22. Berbagai Tipe Himpunan Penyelesaian dari SPLDV



Garis bilangan Himpunan


1. Interval tertutup
{x | a ≤ x ≤ b, x∈R} = [a, b]



2. Interval setengah tertutup
{x | a ≤ x < b, x∈R} = [a, b)


{x | a < x ≤ b, x∈R} = (a, b]



3. Interval terbuka {x | a < x < b, x∈R} = (a, b)



4. Interval setengah garis {x | x ≥ a, x∈R} = [a, ∞)


{x | x > a, x∈R) = (a, ∞)


{x | x ≤ a, x∈R) = (-∞, a]


{x | x < a, x∈R) = (-∞, a)


Simbol ∞ atau -∞ merupakan perluasan ke arah tak berhingga negatif atau tak berhingga positif. Interval yang diperluas sampai tak terhingga dinamakan interval tak hingga. Interval yang titik-titik ujungnya berhingga disebut interval berhingga. Interval berhingga yang memuat satu titik ujung, tetapi tidak memuat titik ujung yang lain disebut interval setengah terbuka atau interval setengah tertutup.

Contoh:

b. 2x + 3y ≥ 6
c. 2x – 3y ≤ 6
d. 2x – 3y > 6

a. 2x + 3y ≤ 6
persamaan garisnya adalah: 2x + 3y = 6
memotong sumbu x di (3,0) dan sumbu y di (0,2)
misal titik (0,0) kita ambil sebagai patokan, kemudian disubstitusi ke pertidaksamaannya, yaitu:
2. 0 + 3. 0 ≤ 6 --> titik (0,0) memenuhi pertidaksamaan
Daerah penyelesaian adalah daerah yang mengandung titik (0,0).
